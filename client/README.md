# Character gen project

TASKS:
- installing needed libraries
  - npm install react-hook-form zod @hookform/resolvers (for forms)
  - npm install aos @types/aos (for scroll animations)
  - npm install @tanstack/react-query (for fetcher)
  - npm install react-hot-toast (for toast messages)

- Installing shadcn components by add and select
- Adding auto pages vite plugin
- Create the page in App.tsx then after that CTRL + Shift and Enter
- Making barrel import files
- Set up your endpoints in src/lib/endpoints.ts
